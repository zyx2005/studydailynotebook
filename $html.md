# css

在HTML中内部链接，一般在<head>里面

```
.hi(选择器){
 color(属性):red(值);
}
```



全局选择器：*

元素选择器：标签名

类选择器：(.)class

ID选择器：(#)属性名

内联选择器：

```
<p style="font-size:50px;color:red;">...</p>
```
